# GameOfLife
консольная игра GameOfLife

Запуск игры:
```zsh
make build
make 1
```

Разные конфигурации игры, подставить вместо 1:

1 - Sliders

2 - Steady figures + oscillators

3 - Longevity R pentomino

4 - Reflector gun - devourer

5 - Random

6 - Train
